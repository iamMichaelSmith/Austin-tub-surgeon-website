# SEO & Content QA Checklist

Use this before publishing new pages or posts.

## On-page essentials
- [ ] One clear primary keyword intent per page
- [ ] Title tag includes keyword + city/service relevance
- [ ] Meta description is unique and compelling
- [ ] H1 is clear and aligned with title intent
- [ ] Image alt text is descriptive and non-spammy

## Local SEO consistency
- [ ] Business name, phone, and service area are consistent
- [ ] Internal links point to relevant service pages
- [ ] Local references (Austin/nearby areas) are accurate

## Technical checks
- [ ] Canonical URL set correctly
- [ ] Page appears in sitemap
- [ ] No broken internal links
- [ ] Build passes (`npm run build`)
