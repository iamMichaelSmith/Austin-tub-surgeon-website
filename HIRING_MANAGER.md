# Hiring Manager Snapshot

## Project purpose
Lead-generation marketing site for a local service business, optimized for SEO and conversion.

## What this project demonstrates
- Practical static-site architecture (Eleventy + Nunjucks)
- Local SEO execution (service pages, city pages, schema, sitemap)
- Conversion workflow design (fast quote funnel with upload flow)
- Documentation and deploy discipline for team handoff

## Business impact
Improves local search discoverability and shortens time-to-quote from inbound visitors.

## Engineering tradeoffs
- Static-first stack for reliability and low hosting complexity
- Form handling delegated to static-host workflows for speed of implementation
- Content-first growth model via local blog clusters
